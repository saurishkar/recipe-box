# react recipe-box

`A Recipe Box built on React and Firebase`

## Stack 
  1. React
  2. React Bootstrap
  3. Firebase API

## Installation

  1. `mkdir react-recipe-box
  2. `cd react-recipe-box
  3. `git clone git@github.com:saurishkar/recipe-box.git
  4. `npm install
  5. `npm start
